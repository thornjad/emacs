#+title:Emacs for Aero
#+author:Jade Michael Thornton

This project is a fork of GNU Emacs which is used to build [[https://gitlab.com/thornjad/aero][Aero Emacs]]. This project contains some customizations for my use cases and is not intended for widespread use.

** Compatibility
*WARNING* this fork is not tested thoroughly, it should not be considered stable.

This project /should/ be compatible with any modern Unix-like system, but is only regularly used on Ubuntu 20.10; other systems are not tested.

** Installation
Full installation information may be found in the =INSTALL= file. But this is the synopsis.

Note that the =-j= option given to =make= is tuned for my 12-core system. By default, it will only use one or two threads in my experience.

#+begin_src sh
./autoconf.sh
make aero-configure
make -j12
sudo make install
#+end_src
